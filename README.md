# 🎨 Color Studio

### Extract stunning color palettes from images & generate harmonious colors.<br/>Everything works blazingly fast in your browser.

## ⚙️ Features

<table>
  <tr>
    <td width="50%">
      <h3>🖼️ Image Palette Extractor</h3>
      Upload or drag & drop an image. Intelligently extracts dominant colors using the HTML5 Canvas API and pixel-clustering algorithms.
    </td>
    <td width="50%">
      <h3>🎲 Random Palette Generator</h3>
      Create beautiful, mathematically harmonious 5-color palettes based on base hues and smooth HSL variations.
    </td>
  </tr>
  <tr>
    <td>
      <h3>🎛️ Live Color Editor</h3>
      Tweak extracted or generated colors in real-time. Full support for RGB/HEX modification and locking specific colors.
    </td>
    <td>
      <h3>📋 One-Click Copy</h3>
      Click any color card to instantly copy its HEX value to your clipboard. Shows a subtle toast notification.
    </td>
  </tr>
  <tr>
    <td>
      <h3>⬇️ Advanced Export</h3>
      Download the entire palette as a sleek PNG strip, copy all HEX codes, or copy them formatted nicely as CSS Variables.
    </td>
    <td>
      <h3>🌈 Gradient Preview</h3>
      Provides a real-time linear gradient preview combining all 5 active palette colors beautifully.
    </td>
  </tr>
</table>

---

## 🔐 Privacy First

> **Your images never leave your device.**
> All color extraction runs purely on your device using Client-Side JavaScript and the HTML5 Canvas.
> No images or data are uploaded to servers.
