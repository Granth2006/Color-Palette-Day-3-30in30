🧠 MVP Prompt — Color Palette Extractor + Generator (HTML, CSS, JS)

Build a fully functional Color Palette Extractor + Generator web app using only HTML, CSS, and vanilla JavaScript (no frameworks).

The app should have a clean, modern UI and be fully responsive.

🎯 Core Features
Image Color Palette Extractor
Allow user to:
Upload an image (drag & drop + file picker)
Extract dominant colors (5–8 colors) from the image using canvas
Display colors as palette cards
Each color card should show:
Color preview
HEX code
Add click-to-copy HEX code
Color Palette Generator
Button: “Generate Palette”
Generate a random harmonious palette (5 colors)
Use logic like:
Random base hue + HSL variations
Display colors same as extractor UI
Edit & Adjust Colors
Allow user to tweak colors:
Hue / Saturation / Lightness sliders OR
Simple color picker
Updating should reflect in real time
Export Options
Copy full palette as:
HEX list
Download palette as:
PNG image (simple palette strip)
🎨 UI/UX Requirements
Minimal, modern design (inspired by Coolors / Adobe Color)
Smooth hover effects
Responsive (mobile + desktop)
Sections:
Upload area
Extracted palette
Generated palette
Add subtle animations (fade / scale)
⚙️ Technical Requirements
Use Canvas API for image processing
Extract colors using:
Pixel sampling OR
Simple clustering approach (like grouping RGB values)
No external libraries unless absolutely necessary
Clean, well-structured JS (functions separated properly)
✨ Bonus (if easy)
“Lock color” feature (like Coolors)
Gradient preview from palette

Copy as CSS variables:

--color-1: #xxxxxx;
📦 Output Format
Provide complete code:
index.html
style.css
script.js
Ensure it works immediately when opened in browser
🚫 Important
Do NOT just create UI — extraction logic must work
Do NOT overcomplicate clustering (keep it MVP-friendly)
Keep performance smooth for large images
